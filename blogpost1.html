NEWBORN IN STYLE

Newborn clothes: sizes Size 000 is intended to fit babies from 0-3 months, and size 00 is for babies from 3-6 months. Some bigger newborns might be ready to go straight into a size 00. You might have to roll up the sleeves, but it won’t be for long. Sizes vary between types of clothes and manufacturers, so it’s worth comparing clothes to other garments you already have, rather than relying only on the size on the label. Babies grow very fast, so it’s best to buy the minimum number of clothes in each size.
